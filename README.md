# JavaScript Exercise 

A collection of JavaScript practice problems and small exercises to strengthen fundamental programming skills.

## 📌 About This Repository

This repository contains multiple JavaScript exercise files focused on:

- Basic syntax
- Functions
- Conditional statements
- Loops
- Array manipulation
- String manipulation
- Problem solving logic

Each file solves a specific problem using clean and simple JavaScript.

---

## 📂 Project Structure

- Each `.js` file represents a separate exercise.
- `index.html` can be used to run scripts in the browser.
- Exercises are beginner-friendly and easy to understand.

---

## 🛠 How to Use

1. Clone the repository:

git clone https://github.com/tar1Qul/JavaScript_exercise.git

cd JavaScript_exercis

node filename.jse
